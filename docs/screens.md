# 画面設計書

## 対象アプリ

Career Tracker

## 1. 画面設計方針

本アプリは、個人の転職活動を横断的に管理する業務アプリである。

PCブラウザでの利用を主目的とし、情報の一覧性・検索性・更新しやすさを重視する。

## 2. 共通レイアウト

### 2.1 全体構成

ログイン後の画面は以下の構成とする。

```text
+--------------------------------------------------+
| Header                                           |
+----------------------+---------------------------+
| Sidebar              | Main Content              |
|                      |                           |
|                      |                           |
+----------------------+---------------------------+
```

### 2.2 Header

表示項目:

* アプリ名
* ログインユーザー情報
* ログアウトボタン

### 2.3 Sidebar

メニュー:

* ダッシュボード
* 転職サービス
* 会社
* 担当者
* 求人
* 応募・選考
* 面談予定
* タスク・期限

### 2.4 Main Content

各画面のメインコンテンツを表示する。

## 4. ログイン画面

### パス

`/login`

### 目的

ユーザーがメールアドレスとパスワードでログインする。

### 表示項目

* メールアドレス入力欄
* パスワード入力欄
* ログインボタン
* エラーメッセージ表示エリア

### 操作

* ログイン
* ログイン成功時は `/dashboard` に遷移
* ログイン失敗時はエラーメッセージを表示

## 5. ダッシュボード

### パス

`/dashboard`

### 目的

転職活動全体の状況を一目で確認する。

### 表示項目

* 応募中件数
* 面談予定件数
* 期限切れタスク件数
* 7日以内の期限タスク件数
* 返信待ち件数

表示項目:

* 日時
* 種別
* 会社名
* 求人タイトル
* オンラインURL

#### 期限が近いタスク

表示項目:

* 期限日
* タイトル
* 種別
* 優先度
* 完了状態

#### ステータス別応募件数

* 気になる
* 応募予定
* 応募済み
* 書類選考中
* カジュアル面談
* 一次面接
* 二次面接
* 最終面接
* 内定
* 辞退
* 不採用

## 24. 共通UI部品

### 24.1 一覧テーブル

使用画面:

* 転職サービス一覧
* 会社一覧
* 担当者一覧
* 求人一覧
* 応募・選考一覧
* 面談予定一覧
* タスク一覧

共通仕様:

* ヘッダー行を表示する
* 行クリックまたは詳細ボタンで詳細画面へ遷移
* 編集ボタンを表示する
* 削除ボタンを表示する
* データがない場合は空状態メッセージを表示する

### 24.2 フォーム

共通仕様:

* 入力項目ごとにラベルを表示する
* 必須項目は視覚的に分かるようにする
* 保存ボタンを表示する
* キャンセルボタンを表示する
* バリデーションエラーを項目下に表示する

### 24.3 ステータスバッジ

対象:

* 応募ステータス
* 優先度
* 志望度
* 完了状態
* 利用状況

### 24.4 削除確認ダイアログ

削除操作時は確認ダイアログを表示する。

表示内容:

* 削除対象名
* 削除すると元に戻せない旨
* キャンセルボタン
* 削除実行ボタン

## 25. 画面遷移方針

### 基本遷移

```text
一覧
 ↓
詳細
 ↓
編集
```

### 新規登録

```text
一覧
 ↓
新規登録
 ↓
保存
 ↓
一覧 または 詳細
```

### 関連情報への遷移

例:

```text
会社詳細
 ↓
関連求人
 ↓
求人詳細
 ↓
応募詳細
 ↓
面談予定詳細
```

## 27. Codexへの実装指示メモ

画面実装は以下の順番で進める。

1. 共通レイアウト
2. ログイン画面
3. ダッシュボード
4. 転職サービス一覧・登録・編集
5. 会社一覧・登録・編集
6. 担当者一覧・登録・編集
7. 求人一覧・登録・編集
8. 応募・選考一覧・登録・編集
9. 選考カンバン
10. 面談予定一覧・登録・編集
11. タスク一覧・登録・編集

初期実装では詳細画面を省略してもよい。

ただし、求人詳細・応募詳細は関連情報を確認する中心画面になるため、優先的に実装する。
